# Personal Website

This is a personal website project Next JS (app directory) app with React Query for fetching data

## Technologies Used

- [Next JS v14](https://nextjs.org/)
- [React Query v5](https://tanstack.com/query/v5)
- [Tailwind CSS](https://tailwindcss.com/)
- [TypeScript](https://www.typescriptlang.org/)

## How To Use

For use this template you can following commands:

```bash
git clone https://github.com/rusdaii/personal-website.git
```

```bash
yarn install
```

## Scripts

### Starts the development server

```bash
yarn dev
```

### Builds the app for production

```bash
yarn build
```

### Runs the built app in production mode

```bash
yarn start
```

### Eslint checking

```bash
yarn lint
```

### Eslint fixing

```bash
yarn lint:fix
```
